# /meta ‚Äî Project Meta Documentation

This directory contains the **foundational and strategic documents** for **Aconcagua: Stone Sentinel**.

It does not host technical documentation or implementation details.  
Instead, it defines the **conceptual framework, vision, high-level decisions, and guiding criteria** of the project.

If you want to understand *what this project is*, *why it exists*, and *from which principles it is being built*, this is the right place to start.

---

## What belongs in `/meta/`?

Documents in `/meta/` serve one or more of the following purposes:

- Define the **intent and meaning** of the project  
- Make **foundational decisions explicit**  
- Establish **principles, limits, and criteria**  
- Provide a **clear entry point** for external readers  
- Support conversations with collaborators, juries, investors, or institutions  

These documents are **public, versioned, and intentionally non-technical**.
